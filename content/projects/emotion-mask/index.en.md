## Overview

Emotion Mask is a 2D platformer built around emotion switching. It was my solo entry for Global Game Jam 2026.

## My role

- Independently handled design, programming, art integration, and sound integration.
- Designed three mask states: calm, happy, and angry, each with different abilities.
- Connected character states with level rules to make state switching the core solution.

## Technical implementation

- Built core character features in Unity and C#, including movement, variable jumps, double jump, dash, wall detection, death, and respawn.
- Used state switching to drive character appearance, movement parameters, dash ability, and scene feedback.
- Used AI tools to help investigate collision, state-machine, and prototype-detail issues.
