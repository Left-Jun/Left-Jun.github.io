    本地工具

    内容编辑

    编辑项目、文章、复盘、计划和关于页面。保存后内容会写回仓库 Markdown 文件。
    打开预览路径

    当前内容的中英文

    选择内容后显示中文 / 英文配对状态。

    基础信息

    这些字段决定页面标题、地址、发布时间、列表摘要和是否发布。

    文件与高级地址

    封面与分类

    图片和视频用于项目卡片/详情页；标签用于筛选和关联。

    资源与项目细节

    资源按钮会自动填入封面字段；JSON 字段主要给项目页补充团队、周期、链接等细节。

    关联内容

    勾选后会在详情页显示“相关项目 / 复盘 / 计划”。

    中英文配对页

    这里按同一个内容键检查中文和英文版本。缺英文时可以先打开中文，再用“创建英文草稿”复制一份基础信息。

    站点配置

    这里控制头像状态、菜单、SEO 默认值等。看不懂时先不要改 JSON 结构，只改文字值。
